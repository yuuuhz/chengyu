# 成语词典

一个纯本地的成语词典网页应用，完全免费，无需服务器。

## 目录结构

```
chengyu/
├── index.html      ← 主程序（单文件，所有功能）
├── start.bat       ← 启动脚本
├── 下载数据.bat    ← 单独下载数据
└── data/
    └── idiom.json  ← 成语数据库（运行 start.bat 自动下载）
```

## 快速开始

**双击 `start.bat`** — 自动下载数据 + 启动服务 + 打开浏览器

浏览器访问：**http://localhost:8080**

---

## 功能说明

| 功能 | 说明 |
|------|------|
| 🔍 成语搜索 | 输入即搜，支持自动补全 |
| 📖 释义/出处 | 来自新华字典数据库（31,648条） |
| 🔗 近义/反义成语 | 由 Gemini AI 生成，**点击可直接跳转** |
| 📝 5条例句 | 由 Gemini AI 生成，展示不同语境 |
| ★ 收藏 | 保存在本地浏览器，无需登录 |
| 🎲 随机成语 | 点击骰子随机探索 |

## Gemini API 设置

近义词、反义词、例句需要 Gemini API Key（**完全免费**）：

1. 访问 https://aistudio.google.com/app/apikey
2. 登录 Google 账号 → 创建 API Key（免费）
3. 点击网页右上角 ⚙️ 图标 → 粘贴 API Key → 保存

**免费额度：** 15次/分钟，每天100万tokens（2个人使用绰绰有余）

**缓存机制：** 每个成语只调用一次 API，结果永久保存在浏览器本地，不重复消耗额度。

## 数据来源

- 成语数据：[pwxcoo/chinese-xinhua](https://github.com/pwxcoo/chinese-xinhua)（31,648条成语）
- 近义/反义/例句：Google Gemini 1.5 Flash（免费额度）
- 收藏数据：浏览器 localStorage（本地）
